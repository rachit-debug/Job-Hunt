# 💼 Job Portal - MERN Stack Application

A modern and responsive **Full Stack Job Portal** built using the **MERN Stack**. The platform allows **Job Seekers** to discover and apply for jobs while enabling **Recruiters** to manage companies, post jobs, and review applications. The application features secure authentication, role-based authorization, Redux state management, and a scalable REST API architecture.

# ✨ Features

### 👨‍💼 Job Seeker

* Create Account & Login
* Secure JWT Authentication
* Update Profile
* Upload Resume
* Browse Available Jobs
* Search & Filter Jobs
* Apply for Jobs
* Track Applied Jobs
* View Company Details

### 🏢 Recruiter

* Register & Login
* Create Companies
* Update Company Details
* Post New Jobs
* Edit/Delete Jobs
* View Applicants
* Accept or Reject Applications

# 🚀 Installation

## Clone Repository

```bash
git clone <your-github-repo-url>
```

## Backend

```bash
cd backend
npm install
npm run dev
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.
